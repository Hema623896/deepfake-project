About This Website

DeepfakeDetect uses a Vision Transformer (ViT) model to analyze images and videos for signs of manipulation. The system stores each analysis in a local database so we can review past predictions and provide a simple explanation for the result.

The explanations are generated automatically based on the model's output: fake predictions mention artifacts commonly found in deepfakes, and real predictions note features consistent with authentic media.
